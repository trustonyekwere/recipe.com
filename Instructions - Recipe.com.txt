** RECIPE.COM **
Use the following guides when working on the Recipe.com web project:

1. Create a folder with the title 'recipe.com'
2. Copy the MaterializeCSS framework (both css & js files, including jquery.js) into the folder
3. Create the index.html file
4. Insert the appropriate meta tags and favicon
5. Download all the images and plae them in the appropriate folder
5. Link all the necessary recources to the index.html file
6. Outline the sections of the web project in the index.html file, using comments, as indicated below:

	i.  	Header
	ii. 	Home
	iii. 	Customers comments
	iv. 	Cakes
	v.	Soups
	vi.	Chicken
	vii.	Footer
